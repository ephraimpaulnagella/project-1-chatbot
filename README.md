# 🤖 Rule-Based Chatbot

## 📌 Project Description
This is a simple chatbot built using Python. It uses if-else conditions and loops to respond to user input.

## 🚀 Features
- Responds to greetings
- Stores user name
- Answers simple questions
- Exit command to stop chatbot

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open terminal
2. Run:
   python chatbot.py

## 💡 Example Inputs
- hi
- my name is John
- what is my name
- how are you
- exit
