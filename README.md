# 🤖 Smart Rule-Based Chatbot

## 📌 Project Overview

This project is a rule-based chatbot developed using Python. It simulates basic human conversation using conditional logic (if-else statements) and runs in a continuous loop to interact with the user.

The chatbot is designed as a foundational Artificial Intelligence system, focusing on control flow, decision-making, and structured interaction.

---

## 🚀 Features

* Responds to greetings (hi, hello, hey)
* Remembers user name
* Tracks user mood
* Provides real-time system time
* Handles exit commands
* Continuous conversation loop
* Modular function-based design

---

## 🛠️ Technologies Used

* Python 3
* VS Code
* Git & GitHub

---

## 🧠 Key Concepts Implemented

* Control Flow (if-else conditions)
* Infinite Loop (while loop)
* String Processing
* Basic State Management (memory of name and mood)
* Function-based program structure

---

## ▶️ How to Run the Project

1. Open terminal in project folder
2. Run the following command:

   python chatbot.py
